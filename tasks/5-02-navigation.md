# Add Navigation UI

## Why this task exists

There are several pages in the web app that comprise the main user journey. They are not interlinked, and there is no way to navigate back and forth between them. This task adds a simple navigation UI (e.g. a header with links) to allow users to move between the main pages of the app.

## Scope

**Included:**

- Add a header component with links to the main pages (as defined in the app's route structure).
- Style the header in line with the rest of the site.

**Out of scope:**

- Implementing any new pages or routes beyond the existing ones.
- Preventing users from navigating to certain pages based on unfinished work ("are you sure you want to leave this page?"). This will be handled in a future task based on user feedback.

## Acceptance criteria

- [ ] Header component exists with links to the main pages
- [ ] Header is styled consistently with the rest of the site
- [ ] Navigation between pages works correctly
